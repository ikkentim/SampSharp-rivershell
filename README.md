SampSharp RiverShell
====================

The _RiverShell_ gamemode in this repository is the same as the version shipped with every copy of the [SA-MP server] package, but has been converted to C#/[SampSharp].

If you find any problems or have questions, submit an issue.

[sa-mp server]: http://sa-mp.com/download.php
[SampSharp]: https://github.com/ikkentim/SampSharp
